Australia States Population Visualization  

Introdution

This map is to show the population(millions), growth rate(%) since 2011 and state area(kmÂ²) metrics of AU states of 2017, 2018 and 2019.

You can tune the following metrics by the control panel at right top corner:

Opacity: The map background opacity

Font Size: The font size of state name and metrics

Font Family: The font family of state name and metrics

Category: The metric you want to show in state position

Year: Which year information you want to know

Highlight: You can highlight specific state on map
