Welcome to the Audio Engineer's Toolbox

This tool was created to help people who are using a graphic equalizer in a live sound environment.
The freeze tool allows you to easily see what frequency is causing the problem.
It also offers the ability to display a canned audio file, which is primarily for evaluating the tool itself without a microphone

Microphone Access
This feature requires you to grant this page access to your microphone. No audio will be recorded, it is only to show the live visualizer.

If you would like to continue, please press continue and then grant access to the microphone on the next screen.
