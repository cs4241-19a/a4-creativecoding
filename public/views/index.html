Conway's Game of Life

Current generation:

The Game of Life operates using rules to determine if a cell lives or dies
The rules are based on the number of living neighbors a cell has:

n=0,1: Any living cell with 0 or 1 neighbors will die, and any dead cell will see no change
n=2: Any living cell will survive, any dead cell will see no change
n=3: Any living cell will survive, any dead cell will be turned into a living cell
n=4+: Any living cell will die, any dead cell will see no change

Each turn in the game is a generation, and the above rules are applied to all cells in the grid (living and dead) at the same time

Press 'h' to open the user interface for the game
The different fields have the following actions:
layout: Select a different starting layout from a premade list (or random)
help: Whether or not to display this message
rules: Whether or not to display the above rules
zoom: A slider for zooming the simulation
stop: Stops the simulation while active
smooth: Applies a trailing effect to cells to smooth the animations and reduce jittering
cellColor: A color selector for the cells on the grid
