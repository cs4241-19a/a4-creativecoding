Sine Wave Visualizer

Documentation Menu

This program uses HTML canvas to display an oscillating sine wave. In the top right corner of this window, there are multiple input sliders that can be used to alter properties of the wave. Values can also be inputted into the textboxes to the right of the sliders. The following properties can be modified:

  • Amplitude: Amplitude or height of the wave
  • Frequency: Lengthens or shortens the period of the wave
  • Phase shift: Horizontal shift of wave
  • Vertical shift: Shifts wave up or down
  • Speed: How fast the wave oscillates
  • Color: Color of the wave
