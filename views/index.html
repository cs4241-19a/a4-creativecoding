Select a bacground image by pasting the image URL into the box, if no image is given it will default to a black screen. Use the zoom slider to set how far the viewpoint will be from the shape. Use the X and Y position sliders to place the shape in the window. Use the dropdown menu to select the shape to be displayed. Click the "Render Shape" button to display the shape.

After Displaying the shape, you can change the rotation speeds around the X, Y, and Z axes using their respective sliders. To modify any other aspects of the image you must repeat the steps described above to create a new shape.
