LifeGrid

This is an implementation of Conway's Game of Life. The visualization involves 3D shapes that get more complex as they are alive for longer.

Using the dialog in the top right corner, you can toggle whether or not the program is running, and also control the rate at which the iterations occur. For fun, you can control the rate at which the objects spin around. You can also enter edit mode.

Edit Mode

When in edit mode, you can click on a cell to toggle if that cell is occupied or not.

Help Screen

You can toggle this help screen in the menu in the top right corner. Enjoy!
