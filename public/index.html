Instructions
everything there is to know about using this application

The application is separated in the three areas Instructions, an example family tree and the custom family tree option.
When you are working with a family tree you can zoom and move it around and use the controls gui to change colors or the width and height of the tree nodes.
In the custom tree form you first have to add the name of one starting person and can then add a spouse. Only after a spouse was added it's possible to also add children. It's also possible to add a second spouse and different children.
Currently only two generations are supported in the custom family tree option.
